# 🥟 Olivia Empanadas & Pizzas

Sitio web oficial de **Olivia**, cadena argentina especializada en empanadas y pizzas desde 2006.

---

## 📖 ¿De qué trata?

Olivia es una cadena de locales que ofrece **empanadas** y **pizzas** de alta calidad en toda la Argentina. El sitio web funciona como punto de encuentro digital para que los clientes conozcan el menú, encuentren sucursales cercanas y realicen pedidos de forma rápida y sencilla.

---

## ✨ ¿Qué contiene la página?

| Sección | Descripción |
|--------|-------------|
| **Inicio** | Hero con acceso directo al menú en PDF (AMBA) |
| **Hacé tu pedido** | Tres opciones: WhatsApp (MOnline), Te Pido y App móvil (iOS/Android) |
| **Menú** | Catálogo completo con filtros por categoría |
| **Galería** | Carrusel con fotos de productos destacados |
| **Novedades** | Promociones y nuevos sabores (Trucha, Choripan, Sin Gluten, Franuí) |
| **Sucursales** | Más de 60 locales con mapa, dirección y teléfono por ciudad |
| **Nosotros** | Historia, calidad y variedad de productos |
| **Franquicias** | Información para sumarse a la #FamiliaOlivia |
| **Contacto** | Formulario para consultas, reclamos, sugerencias y CV |

---

## 🍽️ Productos destacados

- **Empanadas**: Clásicas, especiales, integrales, hojaldradas, árabes y veganas
- **Pizzas**: Tradicionales, veganas y opciones sin gluten
- **Opciones inclusivas**: Productos aptos para celíacos y dietas veganas

## 🚀 Ventajas del sitio

| Ventaja | Detalle |
|---------|---------|
| **Pedidos flexibles** | WhatsApp, Te Pido o App propia según preferencia del cliente |
| **Ubicación fácil** | Buscador de sucursales con mapas integrados de Google Maps |
| **Menú siempre visible** | Descarga del PDF y vista completa en la web |
| **Diseño responsive** | Adaptado a celulares, tablets y escritorio |
| **Múltiples canales** | Contacto por formulario, redes sociales e info de franquicias |
| **Actualizado** | Novedades y nuevos sabores destacados en la portada |
